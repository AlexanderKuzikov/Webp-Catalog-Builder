# WebP Catalog Builder

Утилита для пакетной обработки изображений, извлечения данных о ценах из имен файлов и генерации готового JSON-каталога товаров (e-commerce). Написана на чистом Node.js (zero-dependency).

## Логика работы
1. **Сканирование**: Читает целевую директорию (`source_dir` в конфиге) и находит все `.webp` файлы, содержащие цены в формате `*__<price>.webp`.
2. **Сортировка**: Лексикографически сортирует файлы для сохранения хронологии оригинальных имен (напр. `photo_YYYY-MM-DD_...`).
3. **Генерация дат**: Генерирует массив случайных дат публикации (с учетом только рабочих дней ПН-ПТ и заданных рабочих часов), затем сортирует их от старых к новым.
4. **Переименование и маппинг**: 
   - Назначает инкрементальный SKU (начиная с `start_sku`).
   - Генерирует имя: первый символ SKU отбрасывается, остаток становится номером (напр. SKU `10005` -> `Букет 5`).
   - Калькулирует категорию цены по верхней границе (`<= 2500`, `<= 5000` и т.д.).
   - Переименовывает файлы in-place в формат `<sku>.webp`.
5. **Экспорт**: Формирует JSON массив и сохраняет его в корневой папке утилиты под именем обрабатываемой директории (напр., `photos.json`).

## Установка
Склонируй или скопируй файлы проекта в одну папку. Установка дополнительных npm-пакетов не требуется.
Необходим установленный Node.js (v14+).

## Использование
1. Настрой `config.json`:
   - `source_dir`: Относительный путь к папке с исходными фото.
   - `start_sku`: Стартовый номер (напр. `10001`).
   - `dates`: Диапазон для генерации `date_created` и рабочие часы.
2. Запусти скрипт:
   ```bash
   node index.js

Или через npm:
