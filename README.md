# WebP Catalog Builder

![Node.js](https://img.shields.io/badge/Node.js-v14+-339933?style=flat-square&logo=node.js&logoColor=white)
![Zero Dependencies](https://img.shields.io/badge/dependencies-zero-brightgreen?style=flat-square)
![License](https://img.shields.io/badge/license-ISC-blue?style=flat-square)

Утилита командной строки для пакетной обработки WebP-изображений: извлечение цен из имён файлов, массовое переименование по SKU и генерация JSON-каталога товаров с реалистичными датами публикаций.

---

## Возможности

- Парсинг цены из имени файла по паттерну `*__<price>.webp`
- Пакетное переименование файлов в формат `<sku>.webp`
- Генерация JSON-каталога с полными данными о товарах
- Генерация дат публикации: только рабочие дни (ПН–ПТ), рабочие часы, случайность времени — имитация ручной публикации
- Автоматический расчёт ценовых категорий
- Логирование каждого запуска в отдельный файл

## Конфигурация

Все параметры задаются в `config.json`:

```json
{
  "source_dir": "./photos",
  "start_sku": 10001,
  "category_size": "Средний",
  "dates": {
    "start": "2023-01-01",
    "end": "2023-12-31",
    "work_hours": {
      "start": 9,
      "end": 19
    }
  }
}
```

| Параметр | Описание |
|---|---|
| `source_dir` | Относительный путь к папке с исходными `.webp` файлами |
| `start_sku` | Стартовый SKU (первый символ — служебный префикс) |
| `category_size` | Фиксированное значение поля `category_size` для всех товаров |
| `dates.start` / `dates.end` | Диапазон дат для генерации `date_created` |
| `dates.work_hours` | Рабочие часы для генерации времени публикации |

## Формат входных файлов

Ожидаемый паттерн имени файла:

```
photo_YYYY-MM-DD_HH-MM-SS__<price>.webp
photo_YYYY-MM-DD_HH-MM-SS (N)__<price>.webp
```

Примеры:

```
photo_2026-03-18_05-23-03__4950.webp
photo_2026-03-18_05-47-06 (3)__8830.webp
```

---

## Формат выходного JSON

Файл сохраняется в корне утилиты с именем обрабатываемой директории: `<source_dir>.json`.

```json
[
  {
    "sku": "10001",
    "name_line1": "Букет 1",
    "regular_price": 4950,
    "category_price": "от 2500 до 5000 руб.",
    "category_size": "Средний",
    "date_created": "2023-02-14T10:23:47"
  }
]
```

### Логика ценовых категорий

| Цена | Категория |
|---|---|
| `<= 2500` | до 2500 руб. |
| `<= 5000` | от 2500 до 5000 руб. |
| `<= 7500` | от 5000 до 7500 руб. |
| `> 7500` | свыше 7500 руб. |

### Логика формирования имени

Первый символ SKU является служебным префиксом и отбрасывается при генерации имени:

```
SKU 10001 → "Букет 1"
SKU 10042 → "Букет 42"
SKU 15000 → "Букет 5000"
```

## Логирование

Каждый запуск создаёт отдельный лог-файл в директории `logs/`:

```
logs/run_2026-04-24_11-00-00.log
```

Формат строки:

```
[2026-04-24T11:00:01.000Z] [INFO] [ОК] photo_2026-03-18_05-23-03__4950.webp -> 10001.webp | Цена: 4950
```

В терминал выводится только финальная статистика:

```
=== Обработка завершена ===
Директория: /projects/webp-catalog-builder/photos
Всего валидных файлов: 42
Успешно переименовано: 42
Ошибок: 0
Файл каталога: /projects/webp-catalog-builder/photos.json
Лог: /projects/webp-catalog-builder/logs/run_2026-04-24_11-00-00.log
```
